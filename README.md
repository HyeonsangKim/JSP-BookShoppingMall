# JSP-BookShoppingMall
MVC2 패턴을 이용한 역사책 쇼핑몰

## 주요 기능
* 관리자
    - 회원 관리
    - 카테고리 관리
    - 상품 등록
* 회원
    - 글,댓글 작성
    - 장바구니(담기,삭제)
    - 구매(수량 선택 가능)

## 사용 라이브러리, api
* cos 라이브러리
* JSTL 라이브러리
* DBCP 라이브러리
* 다음 주소 API
* 아임포트 API
